## 활성 전략

### 1. RSI2 역추세 (rsi2_reversal) — 자금 40%

장기 상승추세(MA200) 내 단기 과매도 반등 포착. 승률 중시.

**진입 조건**
- RSI(2) < 10 (과매도)
- 가격 > MA200 (장기 상승추세 내 조정)
- 볼린저밴드 하단 이탈 (추가 확인)
- 거래대금 30일 평균 10억↑

**청산 조건**
- RSI(2) > 70 → 매도
- 손절: ATR×2 (3~7% 범위)
- 보유 10일 초과 → 강제청산

**점수 시스템 (진입 65점↑)**

| 항목 | 배점 | 비고 |
|---|---|---|
| RSI(2) 위치 | 30점 | <5:30, <10:22, <15:11 |
| MA200 위 거리 | 15점 | >20%:15, >10%:11, >0%:7 |
| BB 하단 이탈 깊이 | 15점 | |
| 수급 (외국인+기관 순매수) | 20점 | KRX Dead Cat Bounce 방지 핵심 |
| MRS 맨스필드 상대강도 | 5점 | |
| 5일 하락 후 반등 조짐 | 10점 | |
| 거래대금 증가 | 5점 | 신규 추가 |

### 2. SEPA 트렌드 (sepa_trend) — 자금 50%

미너비니 SEPA 트렌드 템플릿 기반 추세추종.

**진입 조건**
- MA50 > MA150 > MA200 (추세 정렬)
- 가격 > MA50
- 52주 저점 대비 +30% 이상
- 52주 고점 대비 -25% 이내

**청산 조건**
- MA50 하향 이탈 → 매도
- 손절: -5.0% (`default.yml` 기준, 코드리뷰 후 3곳 통일)
- 보유 10일 초과 → 강제청산

**점수 시스템 (진입 60점↑)**

| 항목 | 배점 | 비고 |
|---|---|---|
| 기술적 (SEPA+MA정렬+52주위치+MRS+단기정렬) | ~37점 | |
| 수급 LCI z-score | 20점 | |

## 스크리닝 파이프라인

```
유니버스 (KOSPI500+KOSDAQ150 + 등락률상위50 + 수급상위50)
  ↓
FDR 1년 일봉 + 기술지표 계산 (병렬 10개 동시 / 종목당 타임아웃 10초)
  ↓
거래대금 30일 평균 10억↑ 필터
  ↓
RSI2 / SEPA 조건 분류
  ↓
수급 LCI z-score 계산 (외국인+기관 z-score)
  ↓
밸류에이션 배치 조회 (PER/PBR/ROE)
  ↓
VCP 변동성수축 패턴 탐지
  ↓
전략적 오버레이 (기본점수 50↑ 조건, 최대 +25점 캡)
  - 전문가 패널 최대 +25
  - 수급 추세 최대 +20
  - VCP 최대 +15
  - 복합보너스: 2계층 +5, 3계층 +10
```
